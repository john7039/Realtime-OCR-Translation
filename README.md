# 실시간 OCR 및 번역 시스템 (미완성)
이 프로젝트는 OpenCV, EasyOCR 및 Google Translate를 활용하여 카메라 피드에서 텍스트를 실시간으로 인식하고 번역하는 시스템입니다. 낮은 신뢰도로 인식된 텍스트에 대해서는 이미지 개선 기법을 적용하여 정확도를 높입니다.

## 주요 기능

-   **실시간 텍스트 인식**: 카메라를 통해 실시간으로 텍스트를 감지하고 인식합니다.
-   **이미지 전처리**: OCR 정확도를 높이기 위해 이미지 대비 향상, 노이즈 제거, 이진화 등 OpenCV 기반의 이미지 개선 기법을 적용합니다.
-   **낮은 신뢰도 텍스트 재처리**: 초기 인식 신뢰도가 낮은 텍스트에 대해 이미지 개선 후 재인식을 시도하여 정확도를 향상시킵니다.
-   **다국어 번역**: Google Translate를 사용하여 인식된 텍스트를 사용자가 지정한 언어로 번역합니다.
-   **텍스트 저장**: 인식되고 번역된 텍스트를 `textfile/recognized_text.txt` 파일에 타임스탬프와 함께 저장합니다.

## 설치

이 프로젝트를 실행하기 위해 다음 라이브러리를 설치해야 합니다. 가상 환경을 사용하는 것을 권장합니다.

```bash
# 가상 환경 생성 (선택 사항)
python3 -m venv .venv
source .venv/bin/activate

# 필요한 라이브러리 설치
pip install opencv-python easyocr googletrans==4.0.0-rc1 numpy Pillow scikit-image scipy PyYAML Shapely pyclipper ninja
```

## 사용법

1.  **카메라 권한 설정**: macOS를 사용하는 경우, 시스템 환경설정에서 터미널 또는 IDE에 카메라 접근 권한을 허용해야 합니다. `tccutil reset Camera` 명령어를 터미널에서 실행하여 권한을 재설정할 수도 있습니다.

2.  **스크립트 실행**:
    ```bash
    source .venv/bin/activate # 가상 환경 활성화 (선택 사항)
    python main.py
    ```

3.  **번역 언어 입력**: 프로그램이 시작되면 번역할 대상 언어 코드를 입력하라는 메시지가 나타납니다 (예: `en` for English, `ja` for Japanese, `ko` for Korean). 번역을 원치 않으면 아무것도 입력하지 않고 Enter를 누릅니다.

4.  **텍스트 인식**: 카메라 화면에 텍스트가 있는 사물을 비추면, 실시간으로 텍스트를 인식하고 필요한 경우 개선 및 번역을 수행합니다.

5.  **종료**: `q` 키를 누르면 프로그램이 종료됩니다.

## 파일 구조

-   `main.py`: 주요 OCR 및 번역 로직을 포함하는 메인 스크립트입니다.
-   `Easyocr.py`: 이미지 전처리 및 OCR 관련 함수를 포함하는 모듈입니다. (`main.py`에 통합되어 사용됩니다.)
-   `textfile/`: 인식된 텍스트가 저장되는 디렉토리입니다.
    -   `recognized_text.txt`: 인식된 텍스트가 저장되는 파일입니다.
